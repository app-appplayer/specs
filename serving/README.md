# Serving — Specification

How a bundle is served over the [Model Context Protocol](https://modelcontextprotocol.io): the map from each part of a bundle to the MCP address that carries it, so a server can be coded to it and an AppPlayer-class client connects and runs the result identically to a local bundle. One contract; many servers and clients.
