# Bundle — Specification

The MakeMind ecosystem's bundle format: a self-contained document (manifest metadata + sibling sections — UI, tools, knowledge, settings, wiring, chat, behavior, requires) that packages an application. Built on the [Model Context Protocol](https://modelcontextprotocol.io). One format; many hosts.
